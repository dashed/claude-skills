# Personal Skills Marketplace

> A local marketplace for personal Claude Code skills and plugins.

## Usage

Add this marketplace locally:

```bash
/plugin marketplace add /path/to/claude-skills
```

## Structure

- `.claude-plugin/marketplace.json` - Marketplace manifest
- `plugins/` - (Future) Plugin directories will go here

## Adding Plugins

Edit `.claude-plugin/marketplace.json` to add your plugins and skills.
