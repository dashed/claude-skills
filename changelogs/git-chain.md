# Changelog - git-chain

All notable changes to the git-chain skill in this marketplace will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [1.0.0] - 2025-11-24

### Added
- Initial addition to marketplace for managing chains of dependent Git branches (stacked branches)
- Comprehensive SKILL.md with workflow patterns for stacked PRs and feature chains
- Progressive disclosure structure with references/ directory
- references/rebase-options.md documenting all rebase flags, conflict handling, and recovery
- references/merge-options.md documenting all merge flags, strategies, and reporting options
- references/chain-management.md documenting setup, navigation, and chain organization
- Use cases: stacked PRs, feature chains, review feedback propagation, clean history maintenance
- Version metadata (v1.0.0)
- Author information (Alberto Leal)
- License information (MIT)
- Keywords: git, workflow, branches, stacked, rebase, merge, chain, dependent
- Plugin configuration with skills loading from root
